För att använda programmet ange motsvarande siffra för önskad funktionalitet: 
1 Add Income 
2 Add Expense 
3 View All Transactions 
4 View Balance
5 Remove Transaction
0 Exit

Projektet är en enkel budgethanterare som låter användare lägga till inkomster och uttäkter, visa alla transaktioner, kontrollera balans och ta bort transaktioner. Användaren interagerar med programmet genom att ange siffror som motsvarar olika funktioner.

Reflektioner:

Använding av klasser och metoder gör koden mer organiserad och lättare att underhålla. Genom att kapsla in funktionalitet i klasser kan vi enkelt lägga till nya funktioner.

Den mest utmanande delen var egentligen bara syntaxen.
